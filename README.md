
<center><b>passGen</b>은 간단한 비밀번호 생성기 입니다.</center>

비밀번호에 대한 조건을 설정하면 해당 조건에 알맞은 랜덤값으로 여러개의 비밀번호를 생성 합니다.

## Notice

* JRE 1.7 이상이 설치 되어 있어야 합니다.
* JRE_HOME 환경 변수가 설정 되어 있어야 합니다.
    * 다음이 path에 추가 되어야 함 
		* $JRE_HOME$\bin

## Function
* 조건에 맞는 비밀번호 생성
* 결과를 파일로 저장
